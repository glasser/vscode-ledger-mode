Reconcile Ledger Interface Test Cases

What This Shows

Tests for the interface to the ledger CLI that extracts reconciliation data.

Features tested:

  • Getting list of accounts from ledger file
  • Getting uncleared/pending transactions for a specific account
  • Getting cleared+pending balance for reconciliation
  • Parsing ledger's "emacs" output format

Test cases: 8
