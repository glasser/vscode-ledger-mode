Reconcile Balance Parser Test Cases

What This Shows

Tests for parsing and formatting currency values in the reconciliation view.

Features tested:

  • Parsing currency strings with dollar signs and commas
  • Handling negative amounts
  • Formatting numbers back to display strings with thousands separators
  • Error handling for malformed input

Test cases: 14
