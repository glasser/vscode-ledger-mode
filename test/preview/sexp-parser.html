S-Expression Parser Test Cases

What This Shows

Tests for the S-expression parser used to parse ledger's "emacs" output format.

Features tested:

  • Parsing of atoms (strings, integers, nil)
  • Parsing of lists with nested structures
  • Handling of quoted strings with escape sequences
  • Error handling for malformed input
