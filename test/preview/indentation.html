Indentation Test Cases

What This Shows

This preview shows automatic indentation behavior when pressing Enter on different line types.

Features being tested:

  • Transaction line indentation: Pressing Enter after a transaction line (date-based) adds single space indentation
  • Non-transaction line behavior: Other line types (comments, postings) do not trigger automatic indentation
  • Cursor positioning: Tests work with different cursor positions within lines
  • Indentation highlighting: Leading spaces are highlighted with dotted underlines for visualization
